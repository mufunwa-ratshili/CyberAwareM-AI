# 1. Introduction

## 1.1 Purpose

The purpose of this Software Requirements Specification (SRS) is to define the requirements for CyberAware-AI, an AI-powered phishing detection and cybersecurity awareness platform. This document serves as the foundation for the design, development, testing, deployment, and maintenance of the system.

---

## 1.2 Project Overview

CyberAware-AI is a web-based application designed to help university students recognize phishing attacks while improving their cybersecurity knowledge through interactive learning.

The platform combines intelligent phishing detection with educational resources such as lessons, quizzes, and progress tracking to create an engaging cybersecurity learning experience.

---

## 1.3 Objectives

The objectives of this project are:

- Detect phishing URLs
- Detect phishing emails
- Improve cybersecurity awareness
- Provide interactive learning content
- Track student learning progress
- Generate reports
- Demonstrate secure software engineering practices

---

## 1.4 Intended Users

The system will be used by:

- Students
- Lecturers
- Administrators

---

## 1.5 Development Methodology

The project will follow the Agile Software Development methodology.

Development will be completed in small iterations (sprints), with continuous testing and documentation.
