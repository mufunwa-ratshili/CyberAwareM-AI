# 🛡️ CyberAwareM-AI

> An AI-powered phishing detection and cybersecurity awareness platform built using ASP.NET Core MVC.

---

## 📖 Overview

CyberAwareM-AI is a modern cybersecurity education platform designed to help university students identify phishing attacks while improving their cybersecurity knowledge through interactive learning, AI-assisted phishing detection, quizzes, and progress tracking.

This project is being developed as both a research project and a professional software engineering portfolio.

---

## 🎯 Objectives

- Detect phishing URLs
- Detect phishing emails
- Improve cybersecurity awareness
- Provide interactive learning modules
- Deliver quizzes and assessments
- Track user progress
- Demonstrate secure software engineering practices
